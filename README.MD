# Overview
Generates Morgan Fingerprints and MACCS Keys via RDKit using provided smiles strings in "Lipophilicity" dataset.
Models pair generated values with provided 'exp' values to predict.
Outputs "Deliverables" file with required information.


# Setup
1. Clone Repository 
> git clone https://github.com/Shadowthynightwing/CLSE-572-Lipophilicity-Models.git
> cd HW_5
2. Create Conda Environment
> conda env create -f environment.yml
3. Activate Conda Environment
> conda activate CLSE_572_HW5_Predictions

# Usage
Run main Script as:
> python Run_Models.py
